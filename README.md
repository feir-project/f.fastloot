# f.FastLoot

f.FastLoot is a high-performance auto-pickup solution for Paper and Purpur Minecraft servers. It is designed to be lightweight, customizable, and efficient.

## Features
- Direct-to-inventory item collection to reduce entity lag.
- Customizable block filtering and ignore lists.
- Integrated sound effects for player feedback.
- Full localization support for English and Russian.
- Built-in metrics via bStats (optional).
